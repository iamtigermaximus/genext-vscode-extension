# 🧠 GeNext - AI-Powered Next.js & React TypeScript Helper

[![VS Code Version](https://img.shields.io/badge/VS%20Code-1.85.0+-blue.svg)](https://code.visualstudio.com/)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

GeNext is a powerful VS Code extension that supercharges your Next.js and React development with AI-powered code generation, intelligent analysis, and cloud service integration.

## ✨ Features

### 📦 Code Generators (15+)
- ⚛️ React Components with TypeScript
- 🌐 Next.js API Routes (App & Pages Router)
- 🎣 Custom React Hooks
- 🛠️ Utility Functions
- 🔍 Filter Components
- 📋 List Components (Grid, Carousel, Accordion)
- 🛣️ Dynamic Routes
- 🧪 Test Files
- 📖 Storybook Stories
- 🔗 Context Providers
- 📝 Forms with Validation
- 🪟 Modal Dialogs
- 📊 Data Tables

### 🤖 AI Assistant (DeepSeek)
- 🧠 Code Analysis & Review
- 📝 Automatic Documentation Generation
- 🐛 Bug Detection
- 💡 Code Explanation
- 🔧 Refactoring Suggestions
- 💬 Interactive AI Chat

### ☁️ Cloud Services
- Azure Functions
- AWS Lambda

### 🎨 Styling Helpers
- Tailwind CSS
- Styled Components
- CSS Modules

## 🚀 Quick Start

### Installation
1. Open VS Code
2. Press `Cmd+Shift+X` (Mac) or `Ctrl+Shift+X` (Windows)
3. Search for "GeNext"
4. Click Install

### Configure API Key (for AI features)
1. Get a free API key from [DeepSeek](https://platform.deepseek.com)
2. Open VS Code Settings (`Cmd+,`)
3. Search for "GeNext"
4. Paste your API key
